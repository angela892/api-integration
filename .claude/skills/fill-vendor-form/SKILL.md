# 填廠商申請表單

## 觸發時機

使用者：
- 貼上廠商寄來的申請表（含「商戶名稱」「回調URL」「白名單」等欄位）
- 說「幫我填 XX 廠商申請表」「申請 KFF/KBB/JP 的 FADA」「回填資料」

## 執行步驟

### Step 1 — 釐清必要資訊

如果使用者沒有說明，依序詢問（一次問完，不要分多輪）：

1. **站別**：KFF（金發發）/ KBB（金爸爸）/ JP（日本站）？
2. **代理數量**：
   - KFF：1 個（3代理）/ 2 個（+6代理）/ 3 個（+10代理）
   - KBB：爸幣模式（1個）/ 鑽石模式（1個）/ 兩個都要？
3. **廠商名稱（slug）**：例如 FADA、QT、RSGSW

## 注意事項

- 若廠商的 Callback 需要 Reward URL（如 QT），一併列出
- 不確定的欄位（幣別、商戶名稱細節）先標 `待確認` 並提示使用者補充
- 如果同時申請多代理，每個代理輸出一份表單
- **Hosting Location**：三個站台（KFF / KBB / JP）皆填 `Tokyo, Japan`
- **時區（Time Zone）**：KFF / KBB → `UTC+8`　JP → `UTC+9`
- **官網 / Official Website**：
  - KFF：`https://www.kimfafa.com.tw/`
  - KBB：`https://kbb.tw/official/home.html`
  - JP：`https://www.panahama.jp/`
